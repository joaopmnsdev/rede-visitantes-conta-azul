# Sobre o projeto
Projeto desenvolvido para adaptar a solução Wi-Fi visitantes para a Conta Azul e fornecer uma melhor experiência ao usuário.

# Tecnologias utilizadas
- Javascript
- Html e CSS
- Bootstrap
- vscode
